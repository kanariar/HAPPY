しあわせ貯金箱

日々の小さな「しあわせ」を記録し、思い出を集めるiOSアプリです。


🍎概要🍎

「しあわせ貯金箱」は、あなたが日常で感じた嬉しいこと、誰かへ感謝したことを、

しあわせレベル（金額）をつけて記録できるアプリです。

記録するたびに、かわいいハートのビーズが瓶の中に落ちて、物理演算によってリアルに貯まっていきます。

あなたの「しあわせ」が目に見える形で貯まっていく様子を楽しみながら、さらにポジティブな気持ちになれます。


🍎主な機能🍎

しあわせ貯金（入金）

「しあわせ」な出来事をテキストで記録できます。

5段階の「しあわせレベル」を選択すると、対応する金額が貯金されます。

・ビジュアル貯金箱

記録した「しあわせ」がハート型のマーブルとなり、SpriteKitで描画された瓶の中に貯まっていきます。

マーブルは物理演算に基づいてリアルな動きをします。

・しあわせお返し（出金）

貯まったお金を使って、自分へのご褒美や誰かへのプレゼントなど、「お返し」の記録ができます。

・履歴の確認

これまでの「貯金」と「お返し」の全履歴を一覧で確認できます。

キーワードや日付で履歴を検索する機能も搭載しています。




🍎使用技術🍎

このアプリは、Swiftの勉強も兼ねて、自分が作りたいものを目指して作成しました。

・UIフレームワーク: SwiftUI

・グラフィックス & 物理演算: SpriteKit

・言語: Swift

・プラットフォーム: iOS
