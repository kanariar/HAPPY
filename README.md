しあわせ貯金箱 (Happy Piggy Bank) 💖
日々の小さな「しあわせ」を記録し、可視化するためのiOSアプリです。

✨ 概要
「しあわせ貯金箱」は、あなたが日常で感じた嬉しいことや感謝したことを、金額（しあわせレベル）と共に記録できるアプリです。
記録するたびに、可愛らしいハート型のマーブルが瓶の中にぽとんと落ちて、物理演算によってリアルに積み重なっていきます。あなたの「しあわせ」が目に見える形で貯まっていく様子を楽しみながら、ポジティブな気持ちを育むことができます。
🚀 主な機能
しあわせ貯金（入金）
「しあわせ」な出来事をテキストで記録できます。
5段階の「しあわせレベル」を選択すると、対応する金額が貯金されます。
ビジュアル貯金箱
記録した「しあわせ」がハート型のマーブルとなり、SpriteKitで描画された瓶の中に貯まっていきます。
マーブルは物理演算に基づいてリアルな動きをします。
しあわせお返し（出金）
貯まったお金を使って、自分へのご褒美や誰かへのプレゼントなど、「お返し」の記録ができます。
履歴の確認
これまでの「貯金」と「お返し」の全履歴を一覧で確認できます。
キーワードや日付で履歴を検索する機能も搭載しています。
🛠️ 使用技術
このアプリは、モダンなAppleプラットフォームの技術を学習するために作成されました。
UIフレームワーク: SwiftUI
グラフィックス & 物理演算: SpriteKit
言語: Swift
プラットフォーム: iOS
