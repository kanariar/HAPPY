## 概要

「しあわせ貯金箱」は、あなたが日常で感じた嬉しいこと、誰かへ感謝したことを、しあわせレベル（金額）をつけて記録できるアプリです。

記録するたびに、かわいいハートのビーズが瓶の中に落ちて、物理演算によってリアルに貯まっていきます。
**あなたの「しあわせ」が目に見える形で貯まっていく様子を楽しみながら、さらにポジティブな気持ちになれます。**

Swiftの勉強も兼ねて、自分が作りたいものを目指して作成しました。

---

## 主な機能

* **しあわせ貯金（入金）**
    「しあわせ」な出来事をテキストで記録できます。5段階の「しあわせレベル」を選択すると、その度合いに応じた金額が貯金されます。

* **ビジュアル貯金箱**
    記録した「しあわせ」がハート型のマーブルとなり、SpriteKitで描画された瓶の中に貯まっていきます。マーブルは物理演算に基づいてリアルな動きをするため、視覚的にも楽しめます。

* **しあわせお返し（出金）**
    貯まったお金を使って、自分へのご褒美や誰かへのプレゼントなど、「お返し」の記録ができます。貯めた幸せを循環させる体験を提供します。

* **履歴の確認と検索**
    これまでの「貯金」と「お返し」の全履歴を一覧で確認できます。キーワードや日付で履歴を検索する機能も搭載しており、思い出の振り返りに役立ちます。

## 使用技術

* **SwiftUIとSpriteKitの連携**
    `SpriteView`を使用し、SwiftUIの宣言的なUIの中にSpriteKitの物理演算シーンをシームレスに統合しました。静的なUIパーツと動的なグラフィックスを共存させる設計を行いました。

* **物理演算の実装**
    各ノード（ハートや瓶の枠）に`SKPhysicsBody`を設定し、重力、摩擦、反発係数を調整。単なるアニメーションではなく、物理法則に基づいたリアルな「貯まっていく感触」を再現しました。

* **状態管理によるUI更新**
    `@State`や`@Binding`を活用して、ユーザーが入力した金額やテキストデータを管理。入力アクションが即座にSpriteKit側のシーン（ハートの生成）に反映されるよう、データの受け渡しを実装しました。

* **動的なリスト表示とフィルタリング**
    履歴画面では`List`と`ForEach`を用いてデータを一覧表示。さらにSwift標準の`filter`メソッドや`searchable`修飾子を組み合わせ、膨大な記録からキーワードで検索できる機能を実装しました。

* **レイアウトの工夫**
    `ZStack`を用いて「背景の瓶」と「手前の操作ボタン」を重ねて配置。画面サイズが異なるデバイスでもレイアウトが崩れないよう、スペーシングや配置を調整しました。

## 今後の展望

* **データの永続化**
    アプリを閉じてもデータが消えないように、SwiftData や Core Data を使ってデータをデバイス内に保存する機能。

* **iCloud同期**
    複数のデバイス間でデータを同期し、iPadや新しいiPhoneでもデータを引き継げるようにする機能。

* **ウィジェット対応**
    ホーム画面で現在の貯金額や最新のしあわせを一目で確認できるウィジェットの実装。

* **通知機能**
    毎日の終わりに記録を促すリマインダー通知を送り、習慣化をサポートする機能。

* **デザインの改善**
    ユーザーの好みに合わせたテーマカラーの変更機能や、より洗練されたUI/UXの探求。
